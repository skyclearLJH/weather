# 예상 강수량 방송 그래픽 자동 생성 — 스펙 축적 문서

통보문(예보 텍스트) + 완성 그래픽 쌍을 보고 추출한 규칙 모음.
사용자가 세트를 추가로 제공할 때마다 갱신한다.

## 제작 조건 (사용자 지시)

- 소나기에 의한 예상 강수량 블록은 **모두 제외**
- 어떤 권역이든 **최대치(많은 곳 포함)가 100mm 이상**이면 제작 (사용자 확인 완료)
  - 세트1: 120↑/150↑/250↑ → 제작 O. 세트2의 15일: 최대 5~40 → 제작 X
- 한 통보문에 여러 날짜의 예상 강수량이 있으면 날짜별로 별도 이미지 (조건 충족 시)

## 레이아웃

- 16:9 전체화면. 배경: 비 관련 사진(어두운 톤), 좌측에 사선 어두운 패널
- 좌측 타이틀: "예상 강수량" 큰 흰 글씨 + 밑줄 + 아래 작은 글씨 "(오늘, mm)" — 기간 라벨은 대상일에 따라 (오늘/내일 등)
- 지도: 화면 중앙~우측. 남한 시도(예보구역) 폴리곤, 흰 외곽선, 회색 그림자
- 울릉도·독도: 지도 우상단에 사각 인셋 박스로 별도 표시
- 서해5도: 서해상 작은 섬 폴리곤으로 표시 + 라벨
- 제주: 남쪽에 별도 폴리곤 + 라벨

## 구역 색칠 규칙

- 예보구역(통보문의 구역 단위)별로 단색 채움
- **강수 예보가 없는 권역(소나기만 있는 권역 포함)은 회색**
- "많은 곳 [세부지역] Nmm 이상"이 특정 세부지역에 한정되면:
  그 세부지역을 **한 단계 더 진한 색의 별도 구역**으로 칠하고 라벨은 "N↑"만 단독 표기
  (세트2: 강원중.북부내륙 120↑ 진남색, 충남북부 80↑ 진파랑)
- 시도 내부가 갈리는 구역(강원내륙.산지/동해안, 충남북부/남부, 충북중.북부/남부,
  경기남부 등)은 시군 조합 매핑으로 처리해야 함

## 색상 단계 (관찰値, 상한 또는 '많은 곳' 값 기준으로 진해짐)

| 대표값(최대 기준) | 색 계열 | 관찰 사례 |
|---|---|---|
| ~5(내외) | 거의 흰 하늘 | 5안팎 (세트3 제주·강원동해안) |
| ~20 | 아주 연한 하늘 | 5~20 (세트1·2) |
| ~40 | 연하늘 | 5~40 (세트1·2·3) |
| ~50-70 | 중간 파랑 | 10~50/10~60/10~70 (세트3) |
| ~60-80 | 파랑(로열블루) | 10~60(세트1), 20~80(세트2·3) |
| ~80(많은곳) | 진파랑 | 80↑ (세트2 충남북부) |
| ~100 | 남색 | 30~80(100↑) 세트3 전라 |
| ~120 | 남색(짙은 네이비) | 30~80(120↑) 세트1·2 |
| ~150 | 보라 | 50~100(150↑) 세트1, 세트2 수도권 |
| ~250 | 진보라(거의 검보라) | 50~150(250↑) 세트1 |

- 같은 값이라도 그날 이벤트의 최대 단계에 따라 상대적으로 배색되는 경향
  (세트3처럼 최대 100인 날은 전체가 파랑 계열, 세트1처럼 250인 날은 보라까지 사용)

## 라벨 규칙

- 형식(원칙, 사용자 확정): `최소~최대(많은곳↑)` 예: `30~80(120↑)`, `50~120(150↑)`
  - "많은 곳"이 세부지역에 한정될 때만 그 세부지역을 별도 진한 구역으로 칠하고 `120↑` 단독 표기
  - 세트2 수도권 `50~150↑`, 세트3 전라 `100↑`(30~80 생략)는 디자이너 임의 축약 — 반영하지 않음
- `Nmm 내외` → 라벨 `N안팎` (세트3 제주 `5안팎`)
- 라벨 박스 배경 = 해당 구역 색, 흰색 굵은 숫자
- 기간 예외가 있는 구역은 라벨 위에 작은 흰 글씨로 `(~내일 아침)` 식 표기
  (통보문의 "(수도권, 20일 아침까지)" → "(~내일 아침)"으로 상대화)
  - 세트3: 수도권·강원내륙산지 두 구역에 각각 `(~내일 아침)` 부착 확인
- 라벨은 해당 구역 위 또는 바로 옆, 겹침 없이 배치
- 모든 구역에 라벨을 달지는 않음(작은 값·병합 구역은 생략되기도) — 단 이는 디자이너 재량,
  구현은 통보문 구역별로 라벨을 만들되 배치 단계에서 조정

## 통보문 파싱 패턴 (관찰된 변형)

- 블록: `* 예상 강수량(N일)` ~ 다음 `*` 또는 끝. `* 소나기에 의한 …`은 제외
- 줄 형식: `- (권역) 구역[, 구역]: 범위mm(많은 곳 [세부지역] Nmm 이상)/ 구역: …`
- 기간 한정 변형:
  - 권역 옆: `- (수도권, 20일 아침까지) …`
  - 구역 옆: `강원동해안(19일 낮까지): 5~20mm`
  - 권역 옆 일부 구역만: `- (충청권, 충남북부 20일 새벽까지) …`
- 최대값 변형: `많은 곳 120mm 이상` / `많은 곳 강원남부내륙 100mm 이상`(세부지역 한정)
- 소량 표현: `5mm 내외` (범위 아님 → '5안팎'으로 표기)
- 제외 표현: `광주.전남(남서부 제외), 전북: 30~80mm(많은 곳 100mm 이상)/ 전남남서부: 5~30mm`
  — 구역명 뒤 괄호로 세부지역 제외, 제외된 세부지역은 별도 항목으로 등장
- 빗방울: `0.1mm 미만의 빗방울` — 예상 강수량 아님(본문에만 등장)

## 디자이너 임의 단순화 (구현에 반영하지 말 것 — 사용자 확인 완료)

- 색 단계가 너무 많으면 비슷한 값·인접 구역을 병합하고 범위는 합집합으로 표기
  - 세트2: 충북중.북부 20~60 + 강원내륙.산지 30~80 → 20~80
  - 세트2: 강원북부동해안 10~40 → 5~40 색 구역에 병합(라벨 생략)
  - 세트3: 대구.경북 10~60 + 부산.울산.경남 20~70 → "10~70" 단일 라벨
  - 세트3: 충남서해안 20~80은 수도권 20~80 색과 동일화, 충청내륙 10~60은 경상과 병합
- 라벨 축약: 세트2 수도권 50~120(150↑) → "50~150↑", 세트3 전라 30~80(100↑) → "100↑"
  ← 구현은 항상 원칙 형식 `최소~최대(많은곳↑)` 사용 (사용자 확정)

## 구현 방향 (합의)

- A안: 통보문 자동 파싱 → 초안 그래픽 자동 생성 → 사람이 라벨 위치 등 미세조정(반자동)
- 세부 구역(내륙/산지/동해안/북부/남부) → 시군 조합 매핑 테이블 필요
- 기존 방송모드 지도 인프라(MapLibre + admdongkor 경계) 재활용 가능
